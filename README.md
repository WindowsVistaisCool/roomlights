# roomlights
Tested with an Arduino Uno

## Program List
#### Init Program
0. Loading bar + fade

#### Program Set 1
1. Flutter
2. Rainbow
3. Ocean

#### Program Set 2
1. Back n Forth
2. Rainbow 2
3. 1D Fire

## Button Function
##### Program Button
<strong>Short Press:</strong> Cycle program forward</br>
<strong>Long Press:</strong> Update EEPROM with current program </br>
##### Secondary Button
<strong>Long Press:</strong> Cycle program set forward</br>
